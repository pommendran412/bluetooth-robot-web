# 🤖 Bluetooth Robot Web Controller  

### 🏗️ **Project Overview**
A web-based **Bluetooth-controlled robot interface** with features like:
- **🎮 Joystick Control** for smooth movement
- **🎤 Voice Commands** for hands-free operation
- **🚧 Obstacle Detection Alerts**
- **🔋 Battery Level Display**
- **📱 Mobile-Friendly UI**
